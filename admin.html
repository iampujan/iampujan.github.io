Portfolio Admin Setup

Connect to your GitHub repository to enable dynamic content updates. Changes you make here are committed directly to your repo — no git commands needed.

Go to GitHub → Settings → Developer Settings → Personal Access Tokens
Option A (recommended): Choose Fine-grained tokens → New token → select your iampujan.github.io repo → under Repository permissions set Contents → Read and write
Option B (classic): Choose Tokens (classic) → New token → tick the repo checkbox (full repo access)
Copy the generated token and paste it below
⚠ The token is stored only in your browser's localStorage — never sent anywhere except directly to the GitHub API. If publish fails with "Resource not accessible", your token is missing the Contents write permission (see Step 2).
